# DevOps Public Project Tracker

A simple Python script to simulate building in public for a DevOps engineer. It allows logging progress, challenges, and learnings to a public JSON file, promoting transparency and knowledge sharing.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`.
3. Follow the on-screen prompts to add entries or view the log.
